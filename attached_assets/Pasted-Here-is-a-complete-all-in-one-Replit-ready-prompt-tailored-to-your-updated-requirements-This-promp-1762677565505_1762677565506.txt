## 4. Features & Endpoints

- **Direct Auth:** `/api/auth/login` — sign in instantly as admin/user, set role.
- **Blogs:** `/api/blogs`, `/api/blogs/:id`, `/api/blogs/featured`
- **Categories:** `/api/categories`, `/api/categories/:name`
- **Tags:** `/api/tags`
- **Search:** `/api/search?q=x` (full-text + vector with Milvus)
- **Pagination/Infinite Scroll:** `?page=1&limit=10` on blog list endpoints
- **Related Posts:** `/api/blogs/:id/related` using tag/category/embedding similarity
- **Chatbot (GROQ):** `/api/ai/chat` POST with {query, blog_id}; returns answer/summary/context-aware Q&A
- **Admin Toggle:** front-end switch to sign in as admin/user (no password needed)
- **Content Summarization:** GROQ can summarize current blog, compare with others, give latest facts when asked.

## 7. GROQ GenAI Chatbot Integration

- On blog page, launch floating/chatbot widget.
- For blog-specific Q&A: POST `/api/ai/chat` with user query + blog_id.
- Backend fetches blog context, runs prompt with GROQ for Q&A (current blog context, site knowledge, latest facts).
- Example prompts:
    - “Summarize this blog.”
    - “How is iPhone 17 different from iPhone 16?”
    - “What’s the latest Apple iPhone launch?”
    - “Compare this blog with the last one in technology.”
    - “Show me related blogs on technology.”
